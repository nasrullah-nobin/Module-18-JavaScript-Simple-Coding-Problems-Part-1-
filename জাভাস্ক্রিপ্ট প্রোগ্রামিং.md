জাভাস্ক্রিপ্ট প্রোগ্রামিং — মূল কনসেপ্টের সারাংশ
১. ভেরিয়েবল (Variables)

ডিক্লেয়ারেশন:

var → পুরানো, ব্যবহার না করা ভালো।

let → মান পরিবর্তনযোগ্য ভেরিয়েবল।

let age = 25;
age = 30; // allowed

const → মান পরিবর্তন অযোগ্য (primitive types)

const pi = 3.14;

নোট: const দিয়ে অ্যারে বা অবজেক্ট ডিক্লেয়ার করলে, ভিতরের এলিমেন্ট বা প্রপার্টি পরিবর্তন করা যায়, কিন্তু পুরো অ্যারে বা অবজেক্ট রি-অ্যাসাইন করা যাবে না।

ভেরিয়েবলের টাইপ:

Primitive types: Number, String, Boolean, Undefined, Null

Non-primitive types: Array, Object

২. কন্ডিশনালস (Conditionals)

কম্পেয়ার অপারেটরস:

> : বড়

< : ছোট

> = : বড় বা সমান

<= : ছোট বা সমান

=== : সমান

!== : সমান নয়

লজিক্যাল অপারেটরস:

&& → AND (দুই শর্তই সত্য হতে হবে)

|| → OR (যেকোনো একটি সত্য হলে হবে)

কন্ডিশনাল স্ট্রাকচার:

if (condition) {
// code
} else {
// code
}

অপশনাল: else if

৩. অ্যারে (Array)

ডিক্লেয়ারেশন:

let arr = [1, 2, 3];

ইন্ডেক্সিং:

সব প্রোগ্রামিং ভাষায় ইন্ডেক্স 0 থেকে শুরু।

অ্যারের কোনো আইটেম অ্যাক্সেস বা আপডেট:

arr[0]; // প্রথম আইটেম
arr[1] = 10; // মান আপডেট

মুখ্য মেথডস:

push() → যোগ করা

pop() → শেষ আইটেম রিমুভ

shift(), unshift(), slice(), includes()

sort(), reverse(), join() (প্রয়োজন অনুযায়ী)

৪. লুপ (Loops)

বেসিক লুপ টাইপস:

for → নির্দিষ্ট সংখ্যক iteration

while → condition অনুযায়ী loop

for...of → array বা iterable এর প্রতিটি আইটেমের জন্য

for...in → object এর প্রপার্টি লুপ করার জন্য

কন্ট্রোল স্টেটমেন্টস:

break → লুপ থামানো

continue → বর্তমান iteration skip, পরবর্তী চালানো

৫. অবজেক্ট (Object)

স্ট্রাকচার: Key-Value pair

const person = {
name: "John",
age: 30,
isStudent: true
};

অ্যাক্সেস ও আপডেট:

Dot notation: person.name

Bracket notation: person['age']

Bracket notation with variable: person[varName]

প্রপার্টি টাইপস: Primitive, Array, Object ইত্যাদি

৬. ফাংশন (Function)

ডিক্লেয়ারেশন:

function greet(name) {
return "Hello " + name;
}

কল এবং আর্গুমেন্ট:

greet("Nobin"); // "Hello Nobin"

Return: কোনো মান ফেরত দেওয়ার জন্য

৭. কনস্ট্যান্ট vs লেট (const vs let)
বৈশিষ্ট্য const let
Primitive মান পরিবর্তনযোগ্য? ❌ না ✅ হ্যাঁ
Array/Object ভিতরের মান পরিবর্তনযোগ্য? ✅ হ্যাঁ ✅ হ্যাঁ
নতুন মান দিয়ে re-assignable? ❌ না ✅ হ্যাঁ

সারসংক্ষেপে:

প্রথমে ভেরিয়েবল এবং টাইপ জানা জরুরি।

কন্ডিশনালস ও লজিক্যাল অপারেটরস বেসিক।

অ্যারে এবং লুপ দিয়ে ডাটা ম্যনেজ করা।

অবজেক্ট দিয়ে structured ডাটা রাখার পদ্ধতি।

ফাংশন দিয়ে কোড modular ও reusable করা।
